# Sorting and Searching Algorithms in C#

This repository contains the implementation of various sorting and searching algorithms in C#. The algorithms included are:

- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quicksort
- Counting Sort
- Radix Sort
- Bucket Sort
- Heap Sort
- Shell Sort
- Linear Search
- Binary Search
